Configuration#

MythicDrops has a lot of configuration options. Below is the contents of the itemGroups.yml with inline explanations of what each configuration option does.

  • 6.0.0 (MD 6.1.x)
  • 5.0.0 (MD 6.0.x)
# You can create your own groups here.

# Materials that belong in this item group.

- WOODEN_SWORD
- STONE_SWORD
- IRON_SWORD
- GOLDEN_SWORD
- DIAMOND_SWORD
